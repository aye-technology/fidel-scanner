# üîç Fidel Scanner 

**Fidel Scanner** is a professional, high-performance web application designed to convert images and PDF documents into editable text. Specifically optimized for Ethiopic scripts (**Amharic** and **Tigrinya**) and Latin scripts (**Afaan Oromoo** and **English**), it provides a seamless, high-speed OCR (Optical Character Recognition) experience directly in your browser.

---

## üåü Key Features

* **Multi-Format Support:** Process standard image files (JPG, PNG, GIF) and multi-page PDF documents.

* **Smart Page Selection:** For PDF files, users can select specific pages to scan, saving time and resources.

* **Side-by-Side Dashboard:** Optimized desktop interface featuring two distinct cards for source viewing and result editing, allowing for easy real-time comparison.

* **Local AI Processing:** Unlike most OCR tools, Fidel Scanner performs all AI computations locally on your device. Your sensitive documents never touch a server.

* **Ethiopic Script Optimization:** Utilizes high-quality training models for accurate recognition of Ge'ez characters.

* **Export Options:** Instantly copy extracted text to your clipboard or download it as a `.txt` file.

## üõ†Ô∏è Technical Stack

Built with a modern, lightweight stack to ensure speed and portability:

* **React (v18):** Core framework for state management and UI components.
* **Tesseract.js (v5):** The engine responsible for Optical Character Recognition using deep-learning models.
* **PDF.js:** Mozilla's library for rendering and manipulating PDF documents within the browser.
* **Tailwind CSS:** For a responsive, "utility-first" design that adapts to all screen sizes.
* **Phosphor Icons:** Minimalist and scalable icon set.
* **Noto Sans Ethiopic:** Ensuring Ge'ez characters are rendered with maximum legibility.

## üîí Privacy & Security

**Privacy First.** In an era of cloud processing, Fidel Scanner keeps your data on your machine.

* **No Data Uploads:** Files are processed locally using your device's hardware.
* **Zero Tracking:** We do not track users, store documents, or log extraction results.
* **Offline Support:** Once language models are downloaded, the app can run without an internet connection.

## üöÄ How to Use

1. **Open:** Open the `index.html` file in any modern web browser.
2. **Upload:** Drag and drop an image or PDF into the "Source" box.
3. **Configure:** Choose your target language from the dropdown menu.
4. **Extract:** Click **"Extract Text"**. You can watch the progress in real-time.
5. **Edit & Export:** Refine the extracted text in the result panel, then copy or download it.
